# Tic Tac Toe on Arduino as HW

## Description
Architecture design - MVC. 
The server part executes on HW(microcontroller) and SW(Man Desktop). 
SW is just renderer of HW data.

## Game modes
- Man vs AI (minimax)
- Man vs Man
- AI (minimax) vs AI (minimax)

## Frameworks
- Electron
- Vue JS + Vite
- Platform IO
